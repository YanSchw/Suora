Suora is an open-source C++ Engine for building 3D real-time rendering Applications and interactive Content, especially Video Games.

Overview

Features

Gameplay Programming using C++ and SuoraBuildTool. Everything is devided into Modules.

Full Clean Engine Build within 5 Minutes. Incremental C++ Build (Compile + Editor Startup) in <15 Seconds.

Gameplay Scripting using C#. Extend Native C++ Classes and Functionality with C# and .Net Core 8.

Full C# Build + HotReload in <5 Seconds.

Create custom Gameplay Logic using Visual Scripting. Easy interoperability with C++ using SuoraHeaderTool.
